# Avl-Tree-C

C语言实现的Avl树

### 输出样例

```cpp
// This is adelson velsky and landis tree.
// Init from 1 to 9.
3(3)
|------ 2(1)
|   |------ 1(0)
|   ------ (null)
------ 6(2)
    |------ 4(1)
    |   |------ (null)
    |   ------ 5(0)
    ------ 8(1)
        |------ 7(0)
        ------ 9(0)
```
- input 1 into scanf mode / input -1 quit or other to insert avl
